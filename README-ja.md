# MemoFloat

([English](README.md)/Japanese)

軽量な Chrome 拡張機能で、任意のウェブページ上にフローティングメモを表示できます。  
自由に動かし、サイズを変え、直接入力でき、ページごとに状態を自動保存します。

## Chrome ウェブストア

MemoFloat - Chrome Web Store
https://chromewebstore.google.com/detail/memofloat/pldhpkkjackfhfajdlmgoifcbgpfijcp?authuser=0&hl=ja

## 特長

- **ウェブページ上にフローティング表示されるメモ**
- **ドラッグ可能** — 画面上の好きな位置に移動できる
- **リサイズ可能** — 右下のハンドルをドラッグしてサイズ変更
- **contenteditable 対応** — メモ内に直接テキストを入力
- **ページ単位で自動保存**：
  - メモ内容
  - フォントサイズ
  - 位置
  - サイズ
  - 表示／非表示状態
- **モノトーンのシンプルデザイン**
- **キーボードショートカット対応**
- **1ページにつき1つのメモ**（グローバル共有ではありません）

## 使い方

### メモの表示／非表示
拡張機能アイコンをクリックするか、以下のショートカットを使用します：

- **Mac**: `Alt` + `Shift` + `M`
- **Windows / Linux**: `Alt` + `Shift` + `M`

（`chrome://extensions/shortcuts` で変更可能）

### ボタン

- **A+ / A-** — フォントサイズの拡大／縮小
- **Clear** — メモ内容をクリア
- **× Close** — メモを非表示にする（状態は保存される）

### 自動保存の動作
MemoFloat は以下の操作時に、メモの状態を自動保存します：

- テキストを編集したとき
- メモを移動／リサイズしたとき
- メモを表示／非表示にしたとき
- ページを離れた／再読み込みしたとき

次に同じページを開いたとき、  
**前回メモが表示されていた場合のみ**、前回の状態が復元されます。

## 権限

- `storage` — メモの保存に使用
- `tabs` — アクティブタブへのトグルコマンド送信に必要

トラッキングや解析用の通信は一切ありません。  
MemoFloat はすべてのデータを **ブラウザ内ローカル** に保存します。
